## 1.0.1

- Initial release
- Responsive grid layout with automatic width calculation
- Support for both builder and children patterns
- Configurable spacing and padding
- Fixed and responsive modes
- Animation support
- Loading and empty states

## 1.0.2

- Fixed empty state management
- `emptyIcon`: Customize the empty state icon (default: Icons.grid_off)
- `emptyText`: Customize the empty state message
- `emptyIconSize`: Control icon size (default: 64.0)
- `emptyIconColor`: Customize icon color
- `emptyTextStyle`: Fully customize text styling
